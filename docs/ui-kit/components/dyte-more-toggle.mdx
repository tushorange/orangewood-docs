# dyte-more-toggle

A button which toggles visibility of a more menu.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeMoreMenu: boolean;
}
```

```html
<dyte-more-toggle size="sm"></dyte-more-toggle>
<dyte-more-toggle size="lg"></dyte-more-toggle>
<dyte-more-toggle variant="horizontal"></dyte-more-toggle>
```
