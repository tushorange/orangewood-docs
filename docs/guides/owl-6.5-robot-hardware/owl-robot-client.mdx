## Overview
A Python interface for controlling and receiving data from the OWL robot. The OWL Robot is running a `gRPC` server and a socket interface to provide access to the robot's state data at a frequency of `125Hz`. The `gRPC` server and socket interface allow clients to control and retrieve the robot's state data.
