# dyte-controlbar-button

A skeleton component used for composing buttons.

```html
<dyte-controlbar-button
  label="Flight Mode"
  icon="✈️"
  onclick="alert('flight mode toggled')"
  size="sm"
  class="dyte-el"
></dyte-controlbar-button>
<dyte-controlbar-button
  label="Flight Mode"
  icon="✈️"
  onclick="alert('flight mode toggled')"
  size="lg"
  class="dyte-el"
></dyte-controlbar-button>
<dyte-controlbar-button
  label="Flight Mode"
  icon="✈️"
  onclick="alert('flight mode toggled')"
  variant="horizontal"
  class="dyte-el"
></dyte-controlbar-button>
```
