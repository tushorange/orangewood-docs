# dyte-fullscreen-toggle

A button which toggles full screen mode for any
existing `dyte-meeting` component in the DOM.

```html
<dyte-fullscreen-toggle></dyte-fullscreen-toggle>
<dyte-fullscreen-toggle variant="horizontal"></dyte-fullscreen-toggle>
```
